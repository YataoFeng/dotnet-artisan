Design Principles

KISS First

Simple CRUD doesn't need DDD, MediatR, or CQRS. Scale patterns to problems.

No Repository Wrapping

Use DbContext directly — EF Core is the repository. No useless abstraction layers.

Version-Aware

Guidance adapts to net8.0/net9.0/net10.0/net11.0 automatically.

Framework First

Use what .NET gives you: TimeProvider, ILogger, IHttpClientFactory, System.Text.Json.
