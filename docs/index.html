dotnet-artisan 是一个 Claude Code 插件，让 AI 编码代理能够正确地编写 .NET 代码。

它不是零散的工具集合，而是完整的 .NET 开发智能体系统。决策者先分析需求、捕获领域词汇、设计架构，再路由到对应技能执行——从 API 搭建到调试崩溃，覆盖完整开发生命周期。装完即用，零配置。

工作流程

你的需求 ⟶ 意图检测 ⟶ 决策者编排
▸ 1. 检测 .NET 版本
▸ 2. 需求对齐：提问澄清，捕获领域词汇
▸ 3. 加载 C# 编码规范
▸ 4. 架构设计：调 architect 代理
▸ 5. 拆解领域 ⟶ 并行路由到对应技能
▸ 6. 按需调用专家代理
▸ 7. 工作流优化 + 学习
