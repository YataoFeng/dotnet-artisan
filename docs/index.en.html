dotnet-artisan is a Claude Code plugin that teaches AI coding agents to write correct .NET code.

It's not scattered tools — it's a complete .NET development agent system. A decision-maker analyzes requirements, captures domain vocabulary, designs architecture, then routes to the right skills. From building APIs to debugging crashes, covering the full development lifecycle.

Install and go. Zero config.

How It Works

Your request ⟶ intent detection ⟶ decision-maker
▸ 1. Detect .NET version
▸ 2. Align requirements: ask, capture domain vocabulary
▸ 3. Load C# coding standards
▸ 4. Design architecture (invoke architect agent)
▸ 5. Decompose into domains ⟶ parallel routing
▸ 6. Specialist agents on demand
▸ 7. Workflow + learning
