The 30-Second Rule

Every project generated by this repo MUST be understandable by a fresh AI session in 30 seconds. No chat history needed.

Solution file at root
Never hidden in a subdirectory
Domain project names: OrderManagement.Api
Never Core, Shared, Services
One-line file headers: "Handles fulfillment: validates payment, reserves inventory"
Never "Order handler"
Explicit dependencies: constructor(DbContext, IHybridCache)
Never Service Locator, IServiceProvider
WHY comments: "IServiceScopeFactory because BackgroundService outlives Scoped"
Never "Saves the order"
No generic names
Never Helper, Manager, Utils, Common, Shared

Discovery Pattern

1. Open .slnx вҶ’ See all projects
2. Open Program.cs вҶ’ See DI, middleware, endpoints
3. Open any .cs file вҶ’ Class name + constructor = full story
4. Open appsettings.json вҶ’ See configuration shape

Total: 30 seconds. Full context restored. Zero history needed.
